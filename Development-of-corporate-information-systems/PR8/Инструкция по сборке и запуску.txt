За основу взята практическая работа 6

Распаковать архив.

Запустить psql или pgadmin, создать базу данных или выбрать существующую.

Запустить скрипт DatabaseInit.sql для инициализации и наполнения таблицы.

В файле DBConnection.java в статичных полях URL, user, pass ввести 
ссылку для подключения к выбранной базе данных, имя пользователя и пароль.

В консоли из папки с распакованным архивом ввести команду "mvnw spring-boot:run"
(или "./mvnw spring-boot:run")

Перейти по ссылке "localhost:8080/products/"
(или просто "localhost:8080" - будет сделан редирект на "localhost:8080/products/").
Если используется другой порт то изменить на нужный.

Используется встроенный сервер для брокера сообщений.
сообщения посылаются при создании, изменении, удалении, покупке продукта
сообщения можно просмотреть у пользователя admin во вкладке Messages.
Если возникнет ошибка используемого порта, то поменять порт в application.properties
(broker-url) или использовать настройки для подключения к локальному брокеру


существующие пользователи приложения:

name: user
password: user
может только просматривать продукты и фильтровать их по цене.

name: admin
password: admin
дополнительно может добавлять, удалять, редактировать продукты

новые созданные пользователи также будут обладать только теми правами, что и у user
